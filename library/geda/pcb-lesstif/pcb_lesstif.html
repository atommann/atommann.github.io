使用 PCB 的 Lesstif GUI

Atommann

<Atommann>

　　pcb 除了默认的 GTK GUI 用户界面(GUI)外还可以用 Lesstif，它可以为用户在屏幕上留出更大的使用空间。

　　安装 gd-devel，安装 lesstif 库和头文件，这些在 ./configure --with-gui=lesstif 中都会有提示，安装好需要的库和头文件之后再执行 ./configure 就可以完成配置。

　　PCB 的安装方法在源代码文件夹中的 INSTALL 文件中有详细说明，下面对这个过程作个简要介绍。

  • 执行 ./autogen.sh，这会生成 configure 脚本。

  • 执行 ./configure --with-gui=lesstif，如果这个过程顺利完成，会生成 Makefile

  • 执行 make，注意，在 FreeBSD 上要执行 gmake

  • 执行 cd src 进入源代码目录

  • 执行 ./pcbtest.sh 运行 pcb

图 1. PCB Lesstif 图形用户界面

　　最后，如果你想用 Lesstif GUI，可以执行 make install 将其安装。
