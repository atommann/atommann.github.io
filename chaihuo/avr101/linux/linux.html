简介

在安装了 WinAVR 之后你会发现命令行里多了一些以前不间有的命令，比如 ls touch 等，该教程简单讲解这些命令的使用方法，因为我们在写代码的时候会经常用到它们。

建一个文件夹，然后面里面操作，这个文件夹就是你的游乐场。

ls

ls - 列出目录里的文件和文件夹

当我们要查看当前目录里有什么文件的时候就使用这个命令，在 DOS 里还可以用 dir 命令。

在系统里游走

cd - 切换目录

. 表示当前目录，.. 表示上级目录

尝试 cd . 再执行 pwd，你会发现自己还是在原地。

回到上一级目录 cd .. 再执行 pwd，你会发现自己回到了上级目录。

pwd - 我在哪里

如果你发现自己在 nix 塔里迷路了，就可以用这个命令。

文件操作

mv - 修改文件名

mv 是 move(移动) 的缩写。

mv filea.c fileb.c

rm - 删除文件

rm 是 remove (删除、移除) 的缩写。

自动补齐功能

人们总是喜欢偷懒。 输入前几个字母，按 Tab 键。

重复以前的命令

用向上箭头↑键。

总结

命令 它的含义 它可以干嘛 例子
pwd Present Working Directory 显示你现在所处的目录 pwd
mkdir Make Directory 创建一个文件夹 mkdir hello
ls List 显示文件夹里的内容 ls
